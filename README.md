#  CyberSpace.com | Premium Domain Landing Page

A premium, high-converting, and modern landing page designed for showcasing and selling elite domain assets. This project features a futuristic dark-themed UI, fluid animations, responsive layouts, and interactive components built with pure frontend technologies.

---

##  Features

 **Premium Dark UI**: Built with a sleek, high-end design using the `Plus Jakarta Sans` typography.
 **Interactive 3D Card**: A modern domain metadata display card with glassmorphism and subtle glow effects.
 **Live Countdown Timer**: Built-in countdown functionality targeting specific offer deadlines.
 **Technical Specs Grid**: An organized metadata overview tracking Domain Age, Authority Score, and Registry status.
 **Secure Offer Form**: A fully structured form with real-time input validation formatting for Escrow transactions.
**Fluid Animations & Scroll Reveals**: Smooth entry animations using custom CSS and vanilla JavaScript intersection observers.
 **Fully Responsive**: Optimized for desktop, tablet, and mobile displays.

---

##  Tech Stack

 **HTML5**: Semantic and structured markup for maximum SEO and accessibility.
 **CSS3**: Custom variables (design tokens), Grid/Flexbox layouts, glassmorphic styling, and native animations.
 **JavaScript (Vanilla)**: For modular functionality including scroll-reveal triggers, responsive navigation, form validation, and countdown tracking.
 **FontAwesome v6.4.0**: High-quality vector icons used throughout the interface.
